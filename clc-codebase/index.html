HTML
<nav>

CSS
/* hide hamburger and close button for large screens */
.hamburger, .close{
display: none;
}
/* for small tablets or mobile phones */
@media (max-width: 650px) {
.hamburger{
display: block;
font-size: 3rem;
font-family: 'FontAwesome';
font-weight: bold;
text-decoration: none;
}
nav{
/* hide nav on mobile until hamburger is clicked */
display: none;
}
/* when JS kicks in and adds class, show the menu! */
nav.opened{
background: #b109aa;
display: block;
position: fixed;
padding-top: 20px;
top: 0;
left: 0;
height: 100%;
width: 100%;
text-align: center;
}
nav.opened li{
display: block;
margin: 0 0 10px 0;
}
nav.opened a{
color: white;
text-decoration: none;
}
nav .close{
color: white;
display: block;
font-size: 3rem;
font-family: 'FontAwesome';
font-weight: bold;
position: absolute;
top: 20px;
right: 20px;
text-decoration: none;
}
}
JS
$(".close, .hamburger").click(function(e){
/* prevents page from scrolling to the top */
e.preventDefault();
$('nav').toggleClass('opened');
});

Back to Top button

You may have noticed our button following you in the bottom right hand-side of the screen. Combine this with Smooth Scroll (section above) for a nice effect. You will also notice that it only appears after you scroll past a certain point on the page. You can use the JS below to implement this

Dependencies:
  • Include script for jQuery
HTML
<a href="#">
<img src="images/back-top.png" alt="back to top" />
</a>
CSS
.back-to-top{
display: block;
width: 100px;
height: 100px;
position: fixed;
bottom: 20px;
right: 20px;
}
.back-to-top img{
width: 100%;
}
@media (max-width: 540px){
.back-to-top{
height: 60px;
width: 60px;
}
}
JS
/* this must be outside of the $(document).ready function */
$(window).scroll(function() {
if ($(this).scrollTop() > 160) {
/* 160 = the distance from the top of the page in pixels when you want it to appear */
$('.back-to-top').fadeIn();
} else {
$('.back-to-top').fadeOut();
}
});

Photo Slider

This is sometimes referred to as a carousel or image rotator.

Dependencies:
  • Go to the Flexslider website and click on the green 'Download Flexslider' button. If you are on a PC extract that folder in your downloads.
    • Copy the flexslider.css file and paste it into your project's css folder
    • Copy the jquery.flexslider-min.js file and paste it into your project's js folder
    • Copy the fonts folder and paste it into your project's root folder
    • Your folder structure should now look like this

With these files in place, you will need to edit the flexslider.css file. Flexslider is using an icon font to implement the next and previous arrows on the sides of the slider. We need to point to our fonts folder so that the arrows appear properly.

Go to line 15 of that file and update the paths to the following:
@font-face {
font-family: 'flexslider-icon';
src: url('../fonts/flexslider-icon.eot');
src: url('../fonts/flexslider-icon.eot?#iefix') format('embedded-opentype'),
url('../fonts/flexslider-icon.woff') format('woff'), url('../fonts/flexslider-icon.ttf') format('truetype'),
url('../fonts/flexslider-icon.svg#flexslider-icon') format('svg');
font-weight: normal;
font-style: normal;
}

Other stuff you will need:

HTML
<div class="flexslider">
<ul class="slides">
<li><img src="http://farm9.staticflickr.com/8242/8558295633_f34a55c1c6_b.jpg" /></li>
<!-- Add as many items as you would like in your slider -->
</ul>
</div>
CSS
.flexslider{
max-width: 850px; /* customize this value to your needs */
margin: auto;
}
JS
$('.flexslider').flexslider({
animation: "slide", // other option: "fade"
animationLoop: false, // give the slider a seamless loop
animationSpeed: 600 // Set the speed of animation in milliseconds
});
